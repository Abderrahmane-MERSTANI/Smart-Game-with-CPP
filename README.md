Devine le Nombre - Jeu en C++

Description : -Petit jeu en C++ où l’ordinateur choisit un nombre aléatoire entre 1 et 100. Le joueur doit le deviner. L’ordinateur donne des indices “plus grand” ou “plus petit” jusqu’à ce que le nombre soit trouvé. Le jeu compte le nombre d’essais.

Fonctionnalités

-Génération aléatoire d’un nombre entre 1 et 100.

-Indications pour guider le joueur.

-Compte des tentatives.

Technologies:

-Langage : C++

-Bibliothèques : , ,

Comment jouer:

1.Cloner le dépôt.

2.Compiler le fichier main.cpp avec un compilateur C++ :

  g++ main.cpp -o devine
3.Lancer le jeu :

  ./devine
4.Saisir vos propositions et suivre les indications.

Objectif pédagogique: -Apprendre l’utilisation des boucles, conditions, entrées utilisateur et génération de nombres aléatoires en C++.
